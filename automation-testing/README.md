# Automation Testing - Playwright Framework

This folder contains the Playwright JavaScript automation framework for testing the public website **Automation Exercise**.

## Tech Stack

- Playwright
- JavaScript
- Node.js

## Stability Note

Some tests may appear as flaky because the target public website shows ads and has inconsistent loading behavior.  
Retries and stable locator strategies were added to improve reliability.
